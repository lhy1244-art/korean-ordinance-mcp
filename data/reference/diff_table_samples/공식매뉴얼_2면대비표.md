# 신·구조문 대비표 (2면) 공식 표준 — 경기도의회 운영 절차와 실무 p.196

출처: 「경기도의회 운영 절차와 실무」 매뉴얼 p.196 — 일부개정조례안용 표준 2면 대비표

## 규칙 (그림에서 직접 추출)

1. **2열 표** : 현 행 / 개정안
2. **변경된 부분만 *밑줄*** (한국 자치법규 표준 표기)
3. **변경 안 된 부분은 *대시(`-----`)*** — 가로로 동일 길이 대체
4. **신설 조·항 자체에도 밑줄** (예: `<u>제14조(서면심의 등)</u>`, `<u>(신설)</u>`)
5. **항 전체가 그대로면** 현행 칸 `(생 략)` / 개정안 칸 `(현행과 같음)`

## 표 발췌 (밑줄을 `__...__`로 표시)

| 현 행                                              | 개정안                                                |
| --- | --- |
| 제1조(목적) ------ `__각호__` --------              | 제1조(목적) ------ `__각 호__` --------                |
| 제2조(정의) ------ `__사회__` --------              | 제2조(정의) ------ `__사회복지__` --------             |
| 제5조(준용) ------ <br>`__지방자치법 제36조의__` ----- | 제5조(준용) ------ <br>`__「지방자치법」 제36조의__` ----- |

## 지시문 패턴 (p.195, p.199에서)

이 대비표와 *짝을 이루는* 일부개정조례안 본문의 지시문:

```
○○○○ 조례 일부를 다음과 같이 개정한다.

제1조 중 "각호"를 "각 호"로 한다.
제2조 중 "사회"를 "사회복지"로 한다.
제5조 중 "지방자치법"을 「지방자치법」으로 한다.
```

복합 변경 케이스:
```
안 제3조제7호 중 "사항"을 "사업 계획에 관한 사항"으로 수정하고,
안 제14조제3항을 다음과 같이 신설한다.

③ 제1항에 따라 서면으로 심의·의결하는 경우에는 재적의원 과반수의 찬성을 얻어야 한다.
```
