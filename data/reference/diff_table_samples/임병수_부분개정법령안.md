# 부분개정법령안 신·구조문대비표 작성요령 — 임병수 「법률입안상식」

출처: 임병수 「법률입안상식」, 등록일 2009-01-01 (희연 팀장 제공)

## 작성요령 4규칙 (반드시 준수)

### 1) 대비표는 *현행란*과 *개정안란* 2열로 구분.

### 2) 조문 개정 시 (가장 중요)
- **현행란에는 개정대상 *현행조문* 기재**.
- **항·호·목만 개정되어도 *조제목을 포함한 조 전체*를 기재**한다.
- **개정안란에는 *개정된 후의 개정조문* 기재**.
- **현행 조문과 *같은 부분*은 `…………` 말줄임표로 표시**.
- **현행란의 *개정대상 부분* + 개정안란의 *개정된 부분* 둘 다 *밑줄***.
- **개정대상 *아닌* 항·호·목**:
  - 현행란: `(생 략)`
  - 개정안란: `(현행과 같음)`

### 3) 조·항·호·목 *신설* 시
- **현행란: *빈칸***
- **개정안란: 신설 조·항·호·목 기재 + 밑줄**

### 4) 조·항·호·목 *삭제* 시
- **현행란: 삭제대상 기재 + 밑줄**
- **개정안란: *빈칸***

## 핵심 패턴 정리

| 케이스 | 현행란 | 개정안란 |
|---|---|---|
| 조문 중 일부 문구 수정 | 조 전체 기재, 변경 부분 밑줄 | 변경 안 된 부분은 `………`로, 변경 부분은 텍스트 + 밑줄 |
| 새 항·호 신설 | (빈칸) | 신설 본문 + 밑줄 |
| 항·호 삭제 | 삭제 본문 + 밑줄 | (빈칸) |
| 변경 무관 항 (현행 표시) | `(생 략)` | `(현행과 같음)` |
| 새 조 신설 | (빈칸) | 조 전체 + 밑줄 |
| 조 삭제 | 조 전체 + 밑줄 | (빈칸) |
