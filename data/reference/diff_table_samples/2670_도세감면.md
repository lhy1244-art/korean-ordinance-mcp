# 신·구조문 대비표 표준 양식 — 경기도 도세 감면 조례 일부개정조례안 (의안번호 2670)

출처: 경기도의회 제388회 임시회 (2026.2) 안건 자료 — 김규창 의원 대표발의

## 학습 포인트

이 샘플은 표준 양식의 *정수* — 다음 5가지 패턴이 다 들어있다:

1. **변경 안 된 부분은 대시(`---`)로 시각 대체**, 바뀐 부분만 텍스트로 표시
2. **항 전체가 그대로면 `(현행과 같음)`**, 너무 길어 생략하면 `(생 략)`
3. **신설 행은 `<신   설>`**, **삭제 행은 `<삭   제>`**
4. **항·호 단위로 행 분리** (제4조 ①, ②, 1., 2. 각각 한 행)
5. **2열 구조** — 현 행 / 개정안
